ILauncher
A useful Minecraft launcher

Download

Manage your Minecraft effectively.

Ilauncher is a Minecraft game launcher that help users manage their resources, demonstrate their game, and essentially launch the Minecraft, with great user experience.

Easy Access to Modpack wiki

Ilauncher users can view the modpack available on CurseForge and MinecraftCN.

Users can download the modpacks they favor and launch these modpacks locally.

Modpack information are available on Ilauncher in detail.

Manage all minecraft resources, in one launcher.

It used to be a pain to manage all kinds of resources manually.

Ilauncher has user-friendly interfaces help users manage their resource. Once resources selected/ downloaded, users can use them for their future games.

Manage all minecraft settings, in one place.

Not only high level users can manage game settings, but also novices.

High level players can implement complicated settings using prompt.

New users easily manage settings in the setting page.
